### Synchronized 与 ReenTrantLock
    Synchronized：系统自带，系统自动加锁，解锁
        四种锁状态升级
    ReenTrantLock：需要手动加锁手动解锁
        Condition：不同的等待队列
        CAS + volatile state + 双向队列
